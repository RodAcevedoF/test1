About GitHub

GitHub is a web-based platform used for version control and collaborative software development. It uses Git, a distributed version control system, to help developers track changes, manage projects, and collaborate with others around the world.

  • Repositories: Store and organize your code.
  • Branches: Work on features independently.
  • Pull Requests: Propose and review changes.
  • Issues: Track bugs and enhancements.
